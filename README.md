# Agency starter
This starter template contains a simple agent with a single tool to check the weather.

1. Install dependencies:

```bash
npm install
```

2. Set the openai api key:

```bash
export OPENAI_API_KEY=your_api_key_here
```

3. See if it works:

```bash
npm run agency main.agency
```
